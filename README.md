# Alias-AI

# 🕵️‍♂️ Alias AI Project

משחק "אליאס" חכם שבו הבינה המלאכותית (Google Gemini) מייצרת רמזים יצירתיים למילים סודיות, והשחקן צריך לנחש אותן בכמה שפחות שלבים.

## 👥 חלוקת תפקידים בצוות

* **רפאל (AI - The Brain):** חיבור המערכת לבינה המלאכותית, הנדסת פרומפטים ובקרת איכות הרמזים.
* **הראל בריקמן (The Engine):** ניהול לוגיקת המשחק, מאגר המילים, מערכת הניקוד וניהול התורות.
* **הראל טוסון (The Face):** פיתוח ממשק המשתמש ב-Streamlit, עיצוב הדף ואלמנטים אינטראקטיביים.
* **אריה וורונוב (The Glue & QA):** אינטגרטור המערכת, אבטחת מידע (env.), ניהול ה-Git ובדיקות קצה.

## 🛠 טכנולוגיות
* **Python** - שפת הפיתוח.
* **Streamlit** - ממשק המשתמש.
* **Gemini API** - מנוע ה-AI.
